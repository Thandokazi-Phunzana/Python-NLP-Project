This project focuses on Natural Language Processing (NLP) techniques to perform text classification and text similarity analysis.
It demonstrates how raw textual data can be cleaned, vectorized, and analyzed using machine learning models to extract meaningful insights.

The notebook is designed to be clear, modular, and beginner-friendly, while still following best practices used in real-world NLP projects.

Objectives

Preprocess and clean raw text data
Convert text into numerical representations using NLP techniques
Train and evaluate models for text classification
Measure text similarity between documents
Build a reusable and well-structured NLP pipeline
